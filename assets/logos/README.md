# Ferrovisor Logos

This directory contains all official logos and brand assets for the Ferrovisor project.

## Files

### Vector Logos (SVG)
- `logo.svg` - Main logo in vector format
- `logo-minimal.svg` - Minimal version of the logo
- `logo-modern.svg` - Modern/styled version
- `logo-favicon.svg` - Favicon-optimized version

### Raster Logos (PNG)
- `logo.png` - Main logo in raster format (high resolution)
- `logo-minimal.png` - Minimal version in raster format
- `logo-modern.png` - Modern version in raster format
- `favicon.png` - Favicon in PNG format (256x256)

### Documentation
- `LOGO.md` - Logo usage guidelines and brand information

## Usage Guidelines

### Logo Variants
- **Main Logo**: Use for official documentation, presentations, and marketing materials
- **Minimal Logo**: Use for small sizes, favicons, or minimal design requirements
- **Modern Logo**: Use for contemporary design contexts

### File Formats
- **SVG**: Use for web applications, scalable graphics, and print materials
- **PNG**: Use for applications that don't support SVG or for fixed-size requirements

### Colors
The Ferrovisor logo uses the following primary colors:
- Primary Blue: #2563EB
- Secondary Gray: #64748B
- Accent Colors: As defined in the brand guidelines

## Size Recommendations
- **Favicon**: 32x32px or 16x16px
- **Small Icons**: 64x64px to 128x128px
- **Medium**: 256x256px to 512x512px
- **Large**: 1024x1024px or vector format

For detailed brand guidelines and usage examples, see `LOGO.md`.
